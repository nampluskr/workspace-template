## 4. AI 핵심 제안 및 분석

- **개조식 규칙 `항목명:` 사용 기준 명확화**
  - 초기에는 모든 항목을 `항목명: 설명` 형태로 변환했으나 과잉 레이블링 문제 발생
  - 사용자 피드백으로 규칙 정제: 대상 이름이 없으면 독자가 혼동할 경우에만 `항목명:` 사용
  - `§6 네이밍 규칙`의 `폴더명:`, `루트 레벨 MD 파일:` 등은 유지; `§8 작업 방식`의 번호 항목은 설명만으로 전환

- **`---` 제거 시 공백 처리**
  - 단순 제거 시 연속 빈 줄 발생 — Python regex 로 3개 이상 연속 빈 줄을 2개로 압축하여 처리
